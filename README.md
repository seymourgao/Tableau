
# Tableau模拟某公司销售额与利润分析

利用tableau导入excel数据对数据进行分析、绘图，分析最近几年销售额与利润在不同区域、 销售商品的表现，最后形成可视化数据的输出，对公司未来的销售计划提供建议。

## 依赖
* tableau
